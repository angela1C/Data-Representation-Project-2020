Irish Open Datasets

The database contains some links to dataset resources from where the dataset can be accessed or downloaded. The datasets on the Irish open data portal at data.gov.ie are in a variety of formats including csv, json-stat, JSON, px, API etc. You can search for a link to a dataset resource here.
Enter your search query in the form below.
The data returned comes from a table in the database that was populated by a call to the package_search api at data.gov.ie If there no results returned, you can trigger a call to this api in the External search page. Then come back here to this page.

If you have the actual dataset Id, you can use this to search. For example '00e6465a-313e-41f9-9a1f-18b49e838dca'.

Note that the table may be quite wide as it contains the full url to a dataset.
You can also click on a link to a dataset but please note that this may begin downloading data so proceed with caution!. Some of these datasets are very large.
To delete a record from the database, click the red button for that record. Only a logged in admin user can delete a record from the database.
